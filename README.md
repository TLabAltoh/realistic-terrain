# Realistic Terrain
Add-on for Blender to simulate terrain using directx11 compute shader

## Screenshot
<img src="media/realistic-terrain-cap.gif"></img>

## Requirements
- Windows10
- DirectX11
- Blender using Python 3.10.* (ex. Blender 3.4)
	- Numpy
